# Automobiles Management System

A premium Automobile Management System built with:
- PHP 8
- MySQL
- HTML5/CSS3
- JavaScript
- Bootstrap 5

## Features
- Luxury dealership Home page (slider, live search, filters, gallery, quick view)
- Customer dashboard (bookings, wishlist, requests, notifications)
- Admin dashboard (modules, analytics, reports, exports)
- Security: sessions, CSRF, prepared statements, input validation

## Setup
1) Import `setup/setup.sql`
2) Configure `app/config/config.php`
3) Visit `http://localhost/automates/`

## Project Structure
- `public/` - Front controller + assets
- `app/` - Controllers, Views, Models, Lib
- `api/` - JSON endpoints (vehicle search, wishlist, compare, notifications)
- `setup/` - SQL + setup guide
